## 📐 기술적 접근

### Flutter 프로젝트 생성 명령어
```bash
# app 폴더 생성 및 이동
mkdir app
cd app

# Flutter 프로젝트 생성
flutter create routine_mate \
  --org com.accu.routine \
  --project-name routine_mate \
  --platforms ios,android \
  --description "매일 작은 실천을 통해 큰 변화를 만드는 가장 심플한 습관 트래커"
```
